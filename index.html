SVG badges for all

Have you ever needed to create SVG badges for your GitHub repositories?

Now everything you need to do is use SVG badge!

Using a url like https://svg-badge.appspot.com/badge/kind/label?color=aabbcc you will obtain an SVG image with the text kind followed by label on a background of color #aabbcc.
